# 竞品参考边界

核对日期：2026-07-24。

本项目没有复制即梦或可灵的 UI、提示词或私有实现，只吸收了可验证、适合本地制片工具的产品模式。

| 参考 | 可验证能力/方向 | AI-amnTV 的落法 |
|---|---|---|
| 即梦 | 文生/图生视频、首尾帧控制、偏中文的创作入口 | `first_frame` / `first_last` 模式；中文八段式 prompt；视频付费前检查参考帧 |
| 即梦智能画布 | 生成后继续扩图、局部编辑、消除等迭代方式 | `revise keyframe/video` 单卡局部重做；旧 round/take 永不覆盖 |
| 可灵 | 作为用户指定的对照产品与未来云视频候选 | 通过 `VideoProvider` 保持可替换；未在缺少可核验官方接口资料时伪造适配器 |
| LibTV | OpenAPI 会话、远端 Agent 创作、项目画布、素材上传与结果下载 | 受控外部创作台：本地先落盘、显式上传、同会话续作、手动回收、结果晋升与同量表评测；AI-amnTV 继续掌管审核和交付 |

## 案例：《被裁掉的女孩》

核对日期：2026-07-24。以下分析研究的是可复用的创作与发行机制，不复制角色、剧情、台词、
美术或人物形象。

### 可核验现象

《被裁掉的女孩》由创作者“菲菲飞”连载。抖音公开页的首项互动计数没有在中段快速衰减：
第一集约 55.6 万，第二集约 28.5 万，第五至第七集约 24.3–35.2 万，第十一集约 60.4 万。
这只能作为公开页面快照，不能等同于后台播放量，但足以说明观众并非只围观第一集的 AI 新鲜感。

### 为什么形成持续热度

1. **题眼高度压缩。**“被裁掉”同时指被裁出合照、被行业替代、被城市和关系边缘化。一个可见
   动作承载了完整的身份命题，第一集不需要长篇解释就能建立羞辱、目标和追剧问题。
2. **不是瞬时开挂。**女主通过专业积累、学习、健身、账号经营和一次次选择上升，也持续面对
   人脉、资源、体面和依附的价格。观众获得的是“努力被理解”的现实代偿，而不只是报仇快感。
3. **每集都有可独立传播的情绪结论。**漂亮不等于尊重、被人带路不等于不可替代、机会存在
   隐形价格、梦想靠近后未必浪漫——这些命题既组织当集冲突，也是推荐流里的标题和转发文案。
4. **题材与 AI 视觉相容。**模特、都市和时尚行业本来就允许理想化面孔、广告片质感、精致
   服装与轻微非现实感。题材放大了 AI 的美术效率，同时用旁白、音乐、慢镜头和反应特写降低
   复杂肢体表演压力。
5. **角色按虚拟艺人运营。**分集内容直接关联角色“方桃子”，剧外继续经营角色日常和人物
   关系，使一次观看转成角色关注、CP 讨论和第二季期待。
6. **双层发行。**抖音分集承担钩子、金句、评论和角色运营；完整合集承担连续观看与人物弧
   回收。两种版本不是简单重复，而是分别优化发现与完播。
7. **争议带来第二传播曲线。**撞脸、脸部漂移以及“AI 能否替代演员”的讨论提高搜索与评论，
   但也暴露真人肖像相似、角色版权和授权边界风险，不能把争议本身当作长期产品能力。

### 不能得出的错误结论

- 不是“画面够漂亮就会火”。持续热度依赖剧本、人物弧、配音、分镜、镜头、音乐和剪辑共同
  形成的视听表演。
- 不是“旁白越多越高级”。旁白只有在补充潜台词、时间跨度或主观视角时有效；替代动作和戏剧
  冲突会让作品退化成配图文案。
- 不是“复用热门脸就能造星”。角色辨识度必须来自合法、可锁定的原创设计，不能依赖与真人
  高度相似的生成结果。
- 不是“AI 新鲜感可以支撑一季”。中后段互动能够回升，说明最终仍要靠人物选择和叙事回收。

### 对 AI-amnTV 的直接启示

在立项和剧本关卡加入以下检查：

- 一个可以被拍出来的核心羞辱、损失或不可逆事件
- 一个能贯穿整季、又能被一句话表达的身份命题
- 每集一次有代价的选择，而不只是被动遭遇
- 每集一句来自剧情、可脱离上下文传播的情绪结论
- 角色的剧内成长弧与剧外运营身份
- 分集发现版与连续观看版两套交付策略

视听质量要求详见 [视听表演质量规范](production-quality.md)。
